# Project Blueprint: Body Stretching Guide (Classic)

## Overview
A simple and reliable web guide for body stretching exercises, organized by body parts. Reverted to the stable card-grid interface for better accessibility and reliability.

## Features
- **Categorized Stretches:** Easy navigation through Neck, Shoulders, Back, Wrists, and Legs.
- **Card Interface:** Each exercise is displayed in a clear, readable card.
- **Filtering System:** Instant filtering by body part via navigation buttons.
- **Clean Design:** Fixed light theme with a modern, professional aesthetic.

## File Structure
- `index.html`: Main layout with category filters and grid container.
- `main.js`: Contains stretching data, `<stretch-card>` Web Component, and filtering logic.
- `style.css`: Modern typography and grid layout (Fixed Light Theme).

## Reversion Notes
- Removed 3D Hologram/Three.js due to interaction issues.
- Removed Dark/Light mode switching for a simplified user experience.
